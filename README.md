# CAD 2D — Aggiornamenti pubblici

Questa repository contiene **solo** gli installer Windows di CAD 2D distribuiti
come **Release** di GitHub. L'app installata controlla questa repository
all'avvio e propone gli aggiornamenti automaticamente.

- **Scarica l'ultima versione**: vai nella sezione [Releases](../../releases)
  e scarica l'`.exe` piu' recente.
- Il codice sorgente non e' presente qui: vedi il sito dell'autore.
